# allen-heath-sq-tools

Unofficial Node.js tools for Allen & Heath SQ digital mixing consoles.

## Packages

| Package                                        | Description                                              |
| ---------------------------------------------- | -------------------------------------------------------- |
| [`@allen-heath-sq-tools/api`](packages/sq-api) | TypeScript API for controlling SQ consoles               |
| [`@allen-heath-sq-tools/tui`](packages/sq-tui) | Terminal UI for monitoring and controlling an SQ console |

## Getting Started

```sh
npm install
npm run build
```

## Protocol

The SQ series communicates over a proprietary binary TCP protocol (port 51326) and UDP. This implementation was developed by observing network traffic between an SQ5 and the official MixPad application.

See [`packages/sq-api/FEATURES.md`](packages/sq-api/FEATURES.md) for a list of supported parameters.
